# 至简网格服务仓库
至简网格致力于为中小企业提供各类边缘云上服务。
即使使用一部老旧的安卓机，或者树莓派，也可以实现私有云部署，只要会下载安装手机应用，就可以安装您自己的云服务。
将服务部署在企业自己的私有环境，只要有固定公网IP或实现内网穿透，服务即可全网访问。

## 介绍
此库存放至简网格为中小企业提供的各种服务的源码，以及相应的发布版本。 
至简网格致力于以下四个方面，力图简化中小企业服务的应用、开发、维护成本：


### 1.使用方便，极大降低部署运行成本
企业维持独立的IT开发、运维团队，成本极其高昂，使用至简网格，可以像使用普通手机应用一样安装、设置复杂的企业应用； 
一部**老旧安卓手机、树莓派**等即可运行服务程序，当然也可以在PC上运行，在阿里云、华为云等云可以跨机房多活部署、跨城市备份； 
端侧程序支持安卓/鸿蒙、Windows系统。

### 2.减少代码量
代码即成本，服务的发布包大多不足100K，**帐号、工作流、配置、定时任务、密钥库、oAuth2认证**等基础服务已内置在发布包中，提供了接口级别的认证与鉴权(RBAC、ABAC)，这些极大减少了开发工作量。 
比如，业财一体CRM服务，服务接口与界面实现代码各三千来行，总共七千来行，
而它的功能却并不简陋，实现了客户、联系人管理，订单、服务、回款电子流跟踪，将销售活动与仓储、财务串起来。 
系统内置了工作流服务，工作可以流程化管理与数据按流程授权。

### 3.解决个性化定制问题
企业服务与业务流程紧密相关，千变万化，所以定制是常见现象。
至简网格接口使用json配置以及极少的js片段即可实现，修改极其容易； 
界面部分基于vue+quasar实现，容易理解与定制； 
所有业务代码都以APL协议开源，可以自由定制修改，无法律、商务风险。

### 4.高可靠、高安全性
至简网格按大型互联网企业的可靠性、安全性设计，将可靠性、安全性内置于底层； 
1. 系统可靠-天生支持多实例集群、多机房部署、跨城市备份； 
2. 数据可靠-提供每日数据远程加密备份，即使在地震、火灾等极端情况下，最多只会丢失一天的数据； 
3. 数据安全-在保证易用性的前提下，对数据存储、传输中的安全性做了周密的设计，业务中可以对数据库敏感字段加密，比如电话号码，并且密钥会定期自动更换，不需要做任何额外工作； 
4. 传输安全-服务间通讯、端云之间通讯都使用https协议。

## 软件架构
服务侧内核主要使用java代码以及部分C代码实现，企业服务使用json配置实现接口；
端侧采用网页开发交互界面，简化开发难度，使用体验与原生实现并无差异，windows与安卓端侧保持一致操作体验。


## 安装方法
### 服务侧安装
1.  使用安卓版本服务器，与普通安卓应用一样安装设置，在服务程序的首页点击红色按钮，就启动了服务器； 
2.  如需公有云或私有云集群部署，请联系我们； 
3.  服务程序安装后，打开其中的应用市场中就可以下载所需的服务； 
4.  开通数据云端备份功能，数据每天自动加密备份一次； 
5.  公有云部署，或私有云有固定公网IP，可全球访问； 
6.  私有云无固定公网IP，则使用花生壳之类的动态DNS实现外网穿透，IPv6+UPnP也可以实现外网穿透，但不稳定。 

### 端侧安装
#### 端侧一键安装
下载端侧应用程序安装即可，已提供安卓、Windows版本端侧，不同操作系统的端侧使用方法完全相同。 

#### 端侧两步接入
1. 在“设置”中添加公司，输入公司id、公司接入码
此步骤类似于连接wifi（公司id对应wifi名称，公司接入码对应wifi密码）；
一个终端可以接入多家公司，支持多公司兼职模式； 
2. 点击右上角“登录”按钮输入公司分配的帐号、密码，登入公司系统
此步骤类似接入wifi后登录qq系统，此后就可以操作公司的业务软件。
